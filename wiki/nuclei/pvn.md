# 실방핵 · Paraventricular Nucleus · PVN

## 한 줄 요약

[시상하부](../regions/hypothalamus.md)의 **통합 출력 허브**. ARC에서 받은 신호를
모아 자율신경계와 내분비계로 내보낸다. 섭식 조절뿐 아니라 스트레스·체액 균형까지
출력하는 다용도 핵.

## 위치

[시상하부](../regions/hypothalamus.md)의 위쪽, **제3뇌실 옆에 붙어** 있다.
이름 그대로 "뇌실 옆([para-](../morphemes/para.md) + ventricular)" 핵.

## 기능

- 입력 통합: [ARC](arc.md)의 POMC/CART 뉴런이 보내는 포만 신호를 받아 처리.
- 출력 1 — 자율신경: 교감신경 활동을 조절 → 에너지 소비, 심박, 혈압.
- 출력 2 — 내분비:
  - **CRH** (corticotropin-releasing hormone) 분비 → [뇌하수체](../regions/pituitary.md) 전엽에서 ACTH → 부신피질 코르티솔. **HPA축**의 시상하부 끝.
  - **TRH** → 뇌하수체 → 갑상선축.
  - **옥시토신·바소프레신** (후엽으로 직접 투사) → 수유·자궁수축·수분 보유.
- 섭식 회로에서는 "에너지 소비 ↑, 식욕 억제" 쪽 출력 담당.

## 임상

- PVN 손상 → 시상하부성 비만에 기여 (포만 신호 출력이 약해짐).
- HPA축 이상 (만성 스트레스, 우울)에서 PVN의 CRH 뉴런 활동 변화가 관찰됨.

## 어원·형태소

- [para-](../morphemes/para.md) (Greek) = '옆 / 곁'.
- *ventricular* = '뇌실의' (Latin *ventriculus* '작은 배'). 제3뇌실 옆에 위치.

> 💡 **PVN = 출력 라우터.** ARC가 결정한 신호를 자율신경(즉시) + 호르몬(전신)으로
> 갈라 내보낸다. 섭식만이 아니라 스트레스도 여기를 거쳐 나간다.
