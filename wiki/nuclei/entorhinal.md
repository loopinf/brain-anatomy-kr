# 내후각피질 · Entorhinal Cortex · EC · 內嗅皮質

## 한 줄 요약

[해마](../regions/hippocampus.md)의 **입출력 관문**. 신피질 전역에서 모인 정보가
EC를 거쳐 해마로 들어가고, 해마에서 처리된 정보가 EC를 거쳐 신피질로 다시
나간다. **격자세포(grid cell)**의 고향.

## 위치

- 측두엽 안쪽, [해마](../regions/hippocampus.md) 바로 옆 / 앞.
- 한자 內嗅 = "안쪽 후각" — 후각피질의 일부에서 발달했다는 진화적 기원에서 옴.
  사람에선 후각보단 일반 기억의 관문 역할이 훨씬 큼.

## 기능

### 입출력 관문
- 신피질의 모든 모달리티 정보(시각·청각·체성감각·연합)가 부해마회(parahippocampal)
  → EC → 해마로 흘러간다.
- 해마의 출력도 EC를 거쳐 다시 신피질로 — 시스템 통합의 통로.
- EC 층(layer)별로 역할이 분리:
  - **Layer II** → 천공로(perforant path)로 [DG](dg.md)와 [CA3](ca3.md) 자극.
  - **Layer III** → [CA1](ca1.md) 직접 자극.
  - **Layer V/VI** → 해마 출력 받음, 신피질로 송출.

### 격자세포 (grid cell)
- 2005년 Moser 부부 발견 (2014 노벨상). 내측 EC(MEC)에 사는 뉴런이 **공간을 정삼각형 격자**로 발화 — 동물이 환경을 어디서 움직이든 격자 패턴으로 위치 표상.
- 해마의 [장소세포](ca1.md)가 "여긴 어디"라면, 격자세포는 좌표계 자체. 둘이 합쳐 뇌의 GPS.
- 외측 EC(LEC)는 격자보다 **사물·시간** 정보 표상에 더 관여.

## 임상 — 알츠하이머의 첫 무대

- 알츠하이머 병리(tau 신경섬유엉킴)가 **가장 먼저 EC layer II**에서 시작 (Braak stage I~II).
- 거기서 [해마](../regions/hippocampus.md)로 → 측두 신피질 → 전체 피질 순으로 퍼짐 — 임상 진행과 일치.
- 그래서 **알츠하이머 초기 = EC + 해마 = 새 기억 형성 부전**으로 시작.

## 어원

- *ento-* (Greek '안의') + *rhinal* (코의) = 후각피질의 안쪽 부분.

> 💡 **해마의 관문 + 좌표계.** 격자세포 발견으로 해마-EC가 단순 기억 영역이 아니라
> "공간·시간·상황을 좌표화하는 시스템"임이 드러남.
