# -algia (그리스어, "통증")

## 한 줄 요약

**그리스어 ἄλγος (algos, "통증")** 에서 온 접미사·어근. 의학 진단명에서 "어디가
아프다"를 만드는 거의 유일한 패턴. *an-* (없는)을 붙이면 정반대인 **analgesia**
(무통).

## 도식

{% include diagrams/mor-algia.html %}


## 한 패턴, 많은 용어

- **neur**+**algia** = 신경통.
- **fibro**+**my**+**algia** = 섬유근육통.
- **cephal**+**algia** = 두통 (cephalo- = 머리).
- **gastr**+**algia** = 위통.
- **ot**+**algia** = 귀통증 (oto- = 귀).
- **arthr**+**algia** = 관절통.

## 짝꿍 패턴

- **an**-+**algia** → analgesia = '통증 없음' = 무통·진통. 모든 진통제·하행 진통 회로가 만들어내는 상태.
- **hyper**-+**algesia** → hyperalgesia = '통증 과잉' = 같은 자극에 더 강하게 아픔. 만성 통증에서 [척수 후각](../nuclei/dorsal-horn.md)이 민감해진 결과.
- **allo**-+**dynia** → allodynia = '다른 통증' = 원래 안 아픈 자극(쓰다듬기 등)도 통증으로 느낌. *-dynia*는 *-algia*의 친척 어근.

## noci-와의 차이

[noci-](noci.md)는 해로운 자극의 **물리적 감지**를, *-algia*는 통증의 **경험**을
가리킨다. 그래서:
- nociceptor (수용기) — *noci-*
- neuralgia (경험) — *-algia*
